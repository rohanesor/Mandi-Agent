1. Get Telegram Bot Token (2 min)
   → Open Telegram, find @BotFather
   → Type /newbot
   → Copy token
   → Paste in .env as: TELEGRAM_BOT_TOKEN=...

2. Import Workflows to n8n (2 min)
   → Go to https://rohanesor.app.n8n.cloud
   → Workflows → Import
   → Select 3 JSON files from n8n/workflows/ folder
   → Click Import

   → For each workflow, click the green toggle
   → Copy webhook URL
   → Test with curl (see WEBHOOK_API_REFERENCE.md)
